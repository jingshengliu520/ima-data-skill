# `template-skill`

This is a template for creating a new agent skill. It contains a standard boilerplate for creating python-based tasks.

## Usage Instructions

To use this skill, run the main Python script from the shared repository root:

```bash
python3 skills/template-skill/scripts/main.py
```

### Notes
- Update this `SKILL.md` to describe what your specific skill does.
- Update the `scripts/main.py` with your core Python logic.
